Custom Linked List Library (C)
Overview
This repository contains a robust, custom implementation of a Singly Linked List written in C. Developed as part of my Software Engineering Technology program at Conestoga College, this project serves as a practical demonstration of my proficiency in manual memory management, pointer manipulation, and algorithm efficiency
.
By engineering this library from the ground up, I have reinforced my understanding of how data structures interface with system-level resources, a skill set I maintain with a 3.82 Term GPA
.
Technical Features
Dynamic Memory Allocation: Engineered the library using malloc and free to ensure efficient resource usage and prevent memory leaks
.
Comprehensive Operations: Implemented core data structure functions, including:
Insertion: Head, tail, and arbitrary position insertion.
Deletion: Targeted node removal with proper pointer redirection.
Search & Traversal: Efficient linear search algorithms.
Sorting: Implementation of foundational sorting logic within a linked structure.
Modular Architecture: Following best practices from my Software Quality (A) coursework, the code is highly modular, allowing for easy integration into larger systems
.
Technologies Used
Language: C (Primary focus on pointers and structures)
.
Environment: Developed and tested in a Linux environment using the GCC compiler
.
Tools: Git/GitHub for version control
.
Key Accomplishments
Optimized Memory Overhead: Ensured 100% data integrity during node transitions, preventing common pitfalls such as dangling pointers
.
Validated Performance: Conducted rigorous unit testing to ensure the library handles edge cases, such as operations on empty lists or single-node structures, reflecting the "Quality First" mindset of an aspiring engineer
.
